## Getting Started

### Install

```bash
git clone https://github.com/atiarmridul/atiarmridul.github.io.git
cd atiarmridul.github.io
npm install
```

### Run Development Server

```bash
npm run dev
```

### Build Project

```bash
npm run build
```

### Run Lint

```bash
npm run lint
```

## Deployment

Hosted using GitHub Pages.

```bash
npm run deploy
```
